# MiniGamesWithPython
Some mini games by **python2.7** and **SimpleGUICS2Pygame**

(But you **don't** need to install SimpleGUICS2Pygame)

- Rock-paper-scissors-lizard-Spock
- Guess the number
- Stopwatch The Game
- Pong

## Start up
It's very easy to run:
```bash
$ cd MiniGamesWithPython
$ python game_name.py
```
